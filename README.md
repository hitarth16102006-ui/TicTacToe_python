# Tic Tac Toe Game in Python

A simple command-line Tic Tac Toe game built using Python. Play against another player directly in the terminal and enjoy a classic game experience.

---

## Features

- Two-player gameplay
- Interactive command-line interface
- Input validation
- Detects wins and draws
- Simple and beginner-friendly Python code

---

## Technologies Used

- Python 3

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/tic-tac-toe.git
```

2. Navigate to the project folder:

```bash
cd tic-tac-toe
```

3. Run the game:

```bash
python main.py
```

---

## Gameplay Instructions

- The game board contains positions numbered from 1 to 9.
- Players take turns entering a position number.
- Player **X** goes first, followed by Player **O**.
- The first player to align three marks horizontally, vertically, or diagonally wins.
- If all positions are filled without a winner, the game ends in a draw.

### Example Board Positions

```text
1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9
```
